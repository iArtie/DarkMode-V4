## What it does

DarkMode V4 is a simple mod that turns Geometry Dash's bright interface into a dark theme. It changes the colors of menus, buttons, and UI elements to be easier on the eyes, plus includes a texture pack selector so you can customize the look even further.

## Features

- **Dark UI**: Transforms bright interface elements to darker tones
- **Texture Pack Selector**: Built-in interface to browse and download variants
- **Cross-Platform**: Works on Windows, macOS, Android, and iOS
- **Simple Settings**: Just a few toggle options for customization 

## Manual Installation

1. Install Geode mod loader for Geometry Dash
2. Download DarkMode V4 from the releases page
3. Place the .geode file in your Geode mods folder
4. Launch the game and enjoy the dark theme

## Notes

This README covers the essential information about DarkMode V4. The mod provides dark theme functionality for Geometry Dash's interface and includes a texture pack management system. Version information for texture packs is dynamically pulled from the official GitHub repository index to ensure accuracy.

The texture pack selector system handles downloading and managing the 13 available variants, with current version data fetched from the remote API. Empty variants in the index are marked as "Not available" until they are released.
